Enhancements

Popup

For the popup, we used its properties to display detailed information about a member when clicking on one of the members. This enhances the user experience of the website by making it more interactive and giving the page a cleaner look.

Example: Our team members.

Responsive Graph Design

For the graph design, we make the graph shrink the size when reducing the size of the web page and show another layout when using mobile display. This enhances the user experience of the website by making it easier for mobile users to view.

Example: Our target.

Reference: W3Schools Responsive Guide

Enhanced Navigation with Hover Effect

We've enhanced our website's navigation by adding hover effects to key menus in the header. When users hover over these menus, they're highlighted with a blue bar under menus, making navigation more intuitive. This effect is achieved using CSS hover styles, creating a smooth transition and enhancing the user's browsing experience.

Example: Navigation Bar.

Enhanced Timetable with Hover Effect

We've enhanced our website's timetable by adding hover effects to each specific time. When users hover over these menus, they're highlighted with white text and a blue background when the time is free, and an orange background when the time is busy, making the timetable more interactive. This effect enhances the user's experience.

Example: Timetable.

Hidden Quote

For a more unique design, we added a hidden quote in the about page. When users hover over the empty background, a quote from Steve Jobs will appear. This makes the website more interesting for the user's experience.

Example: Hidden Quote.
